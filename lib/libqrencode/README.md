## libqrencode

Clone or download from:
https://github.com/fukuchi/libqrencode

Copy these source files into this folder:
  qrencode.c / qrencode.h
  qrspec.c   / qrspec.h
  bitstream.c / bitstream.h
  mask.c / mask.h
  mmask.c / mmask.h
  mqrspec.c / mqrspec.h
  qrinput.c / qrinput.h
  rsecc.c / rsecc.h
  split.c / split.h

Then uncomment the `#include "qrencode.h"` line in src/main.cpp.
